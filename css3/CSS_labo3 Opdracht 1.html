Opdracht 1

Doorstreepte regels betekenen dat er een andere soort opmaak werd gemaakt voor eenzelfde element maar die dan bij een andere klasse behoord.



Doorzichtige CSS regels betekenen dat de opmaak doorgegeven werd aan een element door zijn parent-element of het kan ook betekenen dat het zelf door de browser werd toegepast.



Men kan zien of een CSS regel overschreven is door, in element beheer, op computed te drukken. Hier wordt er getoond welke regels de "baas" zijn.
